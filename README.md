Covers Unit 1 (Algorithmic Analysis &amp; Recursion) and Unit 2 (Linear Data Structures) with practical implementations of stacks, arrays, linked lists, queues, and dynamic memory concepts.
## 📘 Data Structures Lab – Unit 1 & Unit 2

This repository contains my practical implementations and assignments for **Data Structures (ETCCDS202)** covering **Unit 1: Foundations & Algorithmic Analysis** and **Unit 2: Linear Data Structures**.

---

## 🔹 Unit 1: Foundations & Algorithmic Analysis

In this unit, I focused on understanding the core concepts of data structures and algorithm efficiency. The implementations include:

* Stack ADT (push, pop, peek, size operations)
* Time complexity analysis using operation counting
* Recursive algorithms:

  * Factorial with call stack trace
  * Fibonacci (Naive vs Memoization)
* Tower of Hanoi problem (recursion and complexity)
* Recursive Binary Search with recurrence relation

📌 Key Learning:

* Understanding **Big-O, Big-Theta, and complexity cases**
* Importance of **recursion and call stack**
* Performance comparison between **naive and optimized approaches**

---

## 🔹 Unit 2: Linear Data Structures

This unit focuses on practical implementation of linear data structures and their real-world behavior.

### Implementations:

* Arrays:

  * 1D operations (insertion, deletion, shifting cost)
  * 2D arrays (matrix traversal, transpose, search)
* Dynamic Array (resize + amortized analysis)
* Linked Lists:

  * Singly Linked List (insert, delete, traverse)
  * Doubly Linked List (advanced operations)
* Stack using Linked List + Parentheses Checker
* Queue using Linked List (O(1) operations)

📌 Key Learning:

* Difference between **static vs dynamic memory**
* Concept of **amortized complexity**
* Efficient data handling using **linked structures**
* Real-world applications of **stack and queue**
